========================================
项目描述
========================================
这是一个使用 Python 语言和 Pygame 库开发的经典四子棋游戏。
游戏支持以下模式：
- 人类 vs 人类
- 人类 vs AI (电脑)
- 限时人类 vs 人类

同时，游戏利用 Tkinter 库实现了简单的消息弹窗。

========================================
运行要求
========================================
1.  已安装 Python (建议使用 Python 3.x 版本)。
2.  已安装 Pygame 库。如果未安装，请打开终端或命令提示符，运行以下命令进行安装：
    ```bash
    pip install pygame
    ```
3.  系统环境中包含 Tkinter (通常 Python 标准安装中已包含)。
4.  （可选）为了更好地显示中文，如果系统默认字体不支持中文，建议安装一个中文字体（如 SimHei 或 Noto Sans SC）。程序会尝试使用系统字体，但可能显示为方框。

========================================
如何运行游戏
========================================
1.  确保您已经安装了 Python 和 Pygame。
2.  游戏代码已经保存为一个 .py 文件（`hw_connect_four.py`）。可使用本地IDE直接编译运行。
3.  也可打开终端、命令提示符或 PowerShell运行。
4.  使用 `cd` 命令导航到您保存 `hw_connect_four.py` 文件的目录。
5.  运行以下命令启动游戏：
    ```bash
    python hw_connect_four.py
    ```
6.  游戏窗口将打开，并首先显示模式选择菜单。

========================================
游戏操作说明
========================================
1. 模式选择菜单：
    -   程序启动后，会显示模式选择界面。
    -   阅读屏幕上的提示，按对应的数字键选择您想玩的游戏模式（1、2、3）。
    -   按 空格键 可以随时退出游戏程序（无论在菜单还是游戏中）。
    -   如果您选择了“人类 vs AI”模式（按 2），会进入一个子菜单让您选择谁先手：按 1 选择玩家先手（红 X），按 2 选择 AI 先手（黄 O）。
2.  游戏进行中：
    -   进入游戏界面后，上方显示当前回合玩家、时间信息（限时模式下显示剩余时间）。
    -   将鼠标移动到棋盘上方的区域，会显示当前回合玩家的棋子提示，指示您即将落子的列。
    -   点击鼠标左键：在您希望落子的列上方点击鼠标左键，棋子将落入该列最下方空位。
    -   目标：先将自己的四个棋子在水平、垂直或两条对角线方向上连成一线即可获胜。
    -   平局：如果棋盘放满所有棋子，且没有玩家获胜，则为平局。
    -   超时 (限时模式)：在限时模式下，任何一方玩家的总用时超过设定上限（默认为 5 分钟），则判该玩家超时，对方获胜。
    -   返回菜单：游戏过程中，随时可以按 空格键 返回主菜单，重新选择模式或退出。
3.  游戏结束：
    -   游戏结束时（胜利、平局、超时），棋盘上方会显示结果消息。
    -   游戏结束后，会在一段时间（默认为 10 秒）后自动返回主菜单。
    -   在游戏结束结果显示期间，您也可以按 空格键 立即返回主菜单。

========================================
注意事项
========================================
*   字符编码： 代码采用 UTF-8 编码。请确保您的终端或运行环境也支持 UTF-8，否则中文文字可能无法正确显示。
*   字体显示： 如果游戏中的中文字符（如菜单、提示信息）显示为方框，说明您的系统缺少程序尝试使用的中文字体（如 SimHei）。程序会回退使用 Pygame 默认字体，但其对中文支持可能不好。
*   弹窗提示： 游戏会弹出一些小窗口提示信息（如“列已满”或游戏结束消息），这些弹窗使用 Tkinter 实现，可能会暂时中断 Pygame 界面直到您关闭弹窗。
*   AI 难度： 本程序的 AI 采用的是一种基于启发式评估的简单策略，智能度有限，适合初级玩家。
